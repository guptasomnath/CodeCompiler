# Code Compiler

This is a Code Compiler, created using Node.js. Here, you can compile your JavaScript, Java, Python, and PHP code.

## Installation

If you want to run this on your local machine, follow the steps below:

**Note:** Ensure that you have installed the necessary compiler for the programming language you wish to compile. For example, for JavaScript, you need Node.js installed on your local machine.

```bash
git clone https://github.com/guptasomnath/CodeCompiler.git
cd CodeCompiler
```

```bash
npm install
node index.js   # For production
npm run dev     # For development
```
